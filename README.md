# Quant Wave Chart

A quantitative trading visualization platform based on Elliott Wave Theory, integrating TradingView Charting Library for real-time chart rendering and analysis.

## Features

- 📊 Real-time K-line chart powered by TradingView Charting Library
- 🌊 Elliott Wave pattern recognition and visualization (1-5 waves, A-B-C corrections)
- 🎯 Multi-resonance zone detection across 8 wave-recognition robots
- 📈 Technical indicators integration (Bollinger Bands, EMA, MACD, RSI)
- 🤖 Automated trading signal generation
- 💹 Backtesting and live trading visualization

## Tech Stack

- TradingView Charting Library (chart rendering)
- Binance API (market data)
- Python (wave recognition engine)
- WebSocket (real-time data feed)
